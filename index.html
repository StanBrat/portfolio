My Skills

I am a full-stack web developer and designer based in Edinburgh. Currently I am a student of Web Development at Edinburgh College where I improve my skills and learn new ones. Web design is my true passion and I'm happy to do it for a living. Except coding I love to hike Munro's and to sail.


I am proficient in HTML and CSS. I can easly use Bootstrap, CSS Flex or CSS Grid. I know how to animate using @keyframes. I am aware of responsive web design RWD and mobile first design, I know how to apply media queries. I'm aware of BEM naming convention for classes. Currently I am practising WordPress for the college project.


I am familiar with Java Scipt and it's library - jQuery. At the college I've learnt some PHP and MySQL, but in the near future I am going to learn JS frameworks like React and Node.

For graphic and video editition I use Adobe Studio - Ps, Ai, Pr and Xd for wireframes.
